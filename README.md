
# URL shortner Backend Server

made using NodeJs and Express




## Tech Stack

**Client:** NextJS and TailwindCSS

**Server:** Node, Express

**Database:** Postgres

**URL shortner method:** base62 conversion

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file


```bash
DATABASE_URI=
DATABASE_PORT=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
```





## Deployment

To deploy this project run

```bash
  npm install
```

```bash
  npm run dev
```
